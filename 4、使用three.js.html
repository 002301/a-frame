4、使用three.js

A-Frame框架是基于three.js开发的，所以支持所有的three.js的API。这一章主要讨论如何访问位于A-Frame框架下的three.js场景、对象和AIP。

  • 访问three.js API
three.js的是A-Frame框架的全局属性

console.log(THREE);
可以直接访问

  • 使用 three.js的Objects
访问three.js的Scene

document.querySelector('a-scene').object3D; // THREE.Scene

并且每一个A-frame实体都能通过 <a-scene> Via.sceneEl访问

document.querySelector('a-entity').sceneEl.object3D; // THREE.Scene

在组件里我们通过 entity (i.e., this.el) 链接scene

AFRAME.registerComponent('foo', {
init: function () {
var scene = this.el.sceneEl.object3D; // THREE.Scene
}
});

  • 访问实体的three.js对象

每一个<a-entity>都有自己的Three.Object3D。具体包括各种类型，实体的根THREE.Group通过 via.object3D访问：

document.querySelector('a-entity').object3D; // THREE.Group

实体还可以组装不同的THREE元素

<a-entity geometry light></a-entity>

实体储存的THREE元素列表还能通过 .object3DMap访问

console.log(entityEl.object3DMap);
// {mesh: THREE.Mesh, light: THREE.Light}

如果通过 .getObject3D(name)访问就会这样

entityEl.getObject3D('mesh'); // THREE.Mesh
entityEl.getObject3D('light'); // THREE.Mesh

现在我们看看这些three.js对象是如果设置的

  • 在实体里设置 object3D 
.setObject3D(name) //创建object3D
AFRAME.registerComponent('pointlight', {
init: function () {
this.el.setObject3D('light', new THREE.PointLight());
}
});
// <a-entity light></a-entity>
首先创造一个光，然后获取
.getObject3D(name)//获取object3D

entityEl.getObject3D('light');

A-Frame 通过 via.el引用一个three.js的object

entityEl.getObject3D('light').el; // entityEl

.removeObject3D(name) //删除object3D 

AFRAME.registerComponent('pointlight', {
init: function () {
this.el.setObject3D('light', new THREE.PointLight());
},
remove: function () {
// Remove Object3D.
this.el.removeObject3D('light');
}
});
