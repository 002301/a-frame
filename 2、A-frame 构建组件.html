2、A-frame 构建组件

在A-Frame框架使用的是 entity-component-system （ECS）工作流。ECS是游戏开发的常见模式，比传统的继承更灵活。

如何理解这个模式：

entity： 实体是一个HTML标签
component：组件作为HTML的重用模块，是实体中的外观、行为或者功能的组合。
sysytem：系统向组件提供全局的服务和管理。例如：事件，交互等等。


  • 标准组件


组件是有以下部分组成：
一个HTML标签   (i.e., <a-entity>).
一些HTML属性
由HTML属性定义的组件属性值，例如：
<a-entity geometry="primitive: sphere; radius: 1.5" material="color: tomato; metalness: 0.7"></a-entity>

geometry：形状
material：材质
还可以添加灯光声音等属性

我们甚至还可以附加其他人创建的第三方组件。 实体 - 组件 - 系统模式实现了极大的灵活性和可扩展性。

可以使用的社区组件总共有三类：
1、template  模板类
2、layout 布局类
3、event-set 事件类

  • template 组件
<a-assets>
<!-- ... -->
<script id="plane" type="text/html">
<a-entity class="link"
geometry="primitive: plane; height: 1; width: 1"
material="shader: flat; src: #cubes-thumb"
sound="on: click; src: #click-sound"></a-entity>
</script>
</a-assets>
然后可以通过模板创建多个实例
<a-entity template="src: #plane"></a-entity>
<a-entity template="src: #plane"></a-entity>
<a-entity template="src: #plane"></a-entity>
但是这样会显示同样的纹理
我们可以使用ES6的字符串插值功能获取变量 (i.e., ${var} format)
<a-assets>
<!-- ... -->
<script id="plane" type="text/html">
<a-entity class="link"
geometry="primitive: plane; height: 1; width: 1"
material="shader: flat; src: ${thumb}"
sound="on: click; src: #click-sound"></a-entity>
</script>
</a-assets>
<!-- ... -->
<!-- Pass image sources to the template. -->
<a-entity template="src: #plane" data-thumb="#city-thumb"></a-entity>
<a-entity template="src: #plane" data-thumb="#cubes-thumb"></a-entity>
<a-entity template="src: #plane" data-thumb="#sechelt-thumb"></a-entity>

  • layout 布局类

因为实体默认位置是0 0 0，所以我们需要布局类排版

<a-entity id="links" layout="layout: line; margin: 1.5" position="-3 -1 -4">
<a-entity template="src: #plane" data-thumb="#city-thumb"></a-entity>
<a-entity template="src: #plane" data-thumb="#cubes-thumb"></a-entity>
<a-entity template="src: #plane" data-thumb="#sechelt-thumb"></a-entity>
</a-entity>

布局类用一个<a-entity>标签承载，设置layout属性实现布局，还可以设置成 grid, circle, 和 dodecahedron.
  • Event-Set 组件类
最后，我们将要在连接上添加一些效果。使用不同事件响应不同的动作。事件列表 cursor events

<a-assets>
<!-- ... -->
<script id="link" type="text/nunjucks">
<a-entity class="link"
geometry="primitive: plane; height: 1; width: 1"
material="shader: flat; src: ${thumb}"
sound="on: click; src: #click-sound"
event-set__1="_event: mousedown; scale: 1 1 1"
event-set__2="_event: mouseup; scale: 1.2 1.2 1"
event-set__3="_event: mouseenter; scale: 1.2 1.2 1"
event-set__4="_event: mouseleave; scale: 1 1 1"></a-entity>
</script>
</a-assets>
使用双下划线的方式区别不同事件，在事件里面添加效果，最后创造出绚丽多彩的VR作品。


  • 编写特定于应用程序的组件

     我们来创建一个通过点击不同的链接来切换360全景图片的组件，叫做 set-image。结构如下：

AFRAME.registerComponent('set-image', {
schema: {
// ...
},
init: function () {
// ...
}
});
接下来定义API需要的功能：
  • 一个事件
  • 需要改变贴图的目标
  • 贴图
  • 动画切换时间

AFRAME.registerComponent('set-image', {
schema: {
on: {type: 'string'},
target: {type: 'selector'},
src: {type: 'string'},
dur: {type: 'number', default: 300}
},
init: function () {
var data = this.data;
var el = this.el;
this.setupFadeAnimation();
el.addEventListener(data.on, function () {
// Fade out image.
data.target.emit('set-image-fade');
// Wait for fade to complete.
setTimeout(function () {
     // Set image.
     data.target.setAttribute('material', 'src', data.src);
}, data.dur);
});
},
setupFadeAnimation: function () {
// Appends an <a-animation> that fades to black.
}
});
