1、 A-Frame 基础

  • 0前言
aframe是Mozilla VR小组创建的webvr制作工具，支持OculusRift和HTCVive。只要使用HTML标签创建3D立体世界。
项目的github地址：https://github.com/aframevr/aframe-boilerplate/
一言不合就上代码

<html>
  <head>
    <script src="https://aframe.io/releases/0.5.0/aframe.min.js"></script>
  </head>
  <body>
    <a-scene>
      <a-box color="#6173F4" opacity="0.8" depth="2" position="0 0 -2"></a-box>
      <a-sphere radius="2" position="1 1 -2"></a-sphere>
      <a-sky color="#ECECEC"></a-sky>
    </a-scene>
  </body>
</html>


这是一个基础框架，包括一个3D场景，一个盒子，一个圆形，和天空盒子。

  • 1 创建场景

<html>
<head>
<script src="https://aframe.io/releases/0.5.0/aframe.min.js"></script>
</head>
<body>
<a-scene>
</a-scene>
<body>
</html>

这就是一个最基础的3D场景框架，场景创建完了就要添加实体.
场景里所有的物体坐标默认都是0 0 0，包括灯光、实体、镜头。
所以默认添加的物体和镜头坐标都是0，是看不到的。

2 添加实体

在标签里可以通过设置css属性改变实体
color="red" //修改颜色
rotation="0 45 45" //旋转
scale="1 0 1" //改变大小
src="#id" //设置纹理
position="0 0 0"//设置坐标
width/height：设置宽/高

盒子：

<a-box color="red" rotation="0 45 45" scale="2 2 2"></a-box>

平面：

<a-plane rotation="0 0 0" ></a-plane>


天空盒子：

<a-sky color="#222"></a-sky>


添加纹理管理系统：
添加纹理的步骤：
1 把<a-assets>添加到scene里面。
2 把图片标签<img>添加到<a-assets>
3 给img纹理设置ID
4 在实体上添加（src="#id")

<a-assets>
    <img id="boxTexture" src="https://i.imgur.com/mYmmbrp.jpg">
</a-assets>


设置纹理

<a-sky src="#boxTexture" color="#222"></a-sky>


添加地面：

<a-plane src="#boxTexture" rotation="-90 0 0" width="30" height="30"></a-plane>

添加纹理的时候可以通过设置 repeat="10 10" 来设置平铺的重复次数，在方向X和Y上重复多少次，在3D里面又叫UV贴图。

<a-plane src="#boxTexture" rotation="-90 0 0" width="30" height="30" repeat="10 10"></a-plane>


调整照明
A-frame框架默认一个环境光，在我们添加<a-light>标签后就可以自定义环境光。
框架里环境光和点光源可以同时添加。
A-frame里面有两种照明方式 type="ambient" 和 type="point" 环境光和点光源
<a-light type="ambient" color="#445451"></a-light>
<a-light type="point" intensity="2" position="2 4 4"></a-light>

设置镜头
添加镜头的时候最好把z轴向后移动一下，这样默认的实体就是按照0 0 0坐标对齐也能看到了。
<a-camera position="0 0 5" ></a-camera>

  • 添加动画
<a-animation> 这个标签实现了实体的动画，但是后期会有改动，可以直接在实体上添加动画二去掉标签。
<a-animation attribute="position" to="0 2.2 -5" direction="alternate" dur="2000"
repeat="indefinite"></a-animation>

attribute 要运动的属性
to ：设置运动后的数值
direction="alternate" //运动方式 alternate循环
dur：设置运动持续的时间
repeat="indefinite" 重复次数，无限循环


添加交互

因为大多数人没有VR设备，所以框架提供了一个内部光标组件，模拟VR指针。这个光标组件需要添加到 <a-camera>里面

<a-camera>
    <a-cursor></a-cursor>
  </a-camera>


这个光标组件包括 mouseenter, mouseleave 和 click 事件。
可以通过几种方式实现
这种方式是标准的js方式

<script>
  var boxEl = document.querySelector('a-box');
  boxEl.addEventListener('mouseenter', function () {
    boxEl.setAttribute('scale', {x: 2, y: 2, z: 2});
  });
</script>


还可以这样，用a-frame的方法

<a-box color="#FFF" width="4" height="10" depth="2"
      position="-10 2 -5" rotation="0 0 45" scale="2 0.5 3"
      src="#texture">
  <a-animation attribute="position" to="0 2.2 -5" direction="alternate" dur="2000"
    repeat="indefinite"></a-animation>
  <!-- These animations will start when the box is looked at. -->
  <a-animation attribute="scale" begin="mouseenter" dur="300" to="2.3 2.3 2.3"></a-animation>
  <a-animation attribute="scale" begin="mouseleave" dur="300" to="2 2 2"></a-animation>
  <a-animation attribute="rotation" begin="click" dur="2000" to="360 405 45"></a-animation>
</a-box>


  • 添加声音
在素材标签里添加声音

<a-scene>
  <a-assets>
    <audio src="https://cdn.aframe.io/basic-guide/audio/backgroundnoise.wav" autoplay
      preload></audio>
  </a-assets>
  <!-- ... -->
</a-scene>

还可以设置声音的位置，毕竟是VR吗对吧。

<a-scene>
  <!-- ... -->
  <a-sound src="https://cdn.aframe.io/basic-guide/audio/backgroundnoise.wav" autoplay="true"
    position="-3 1 -4"></a-sound>
  <!-- ... -->
</a-scene>
